## Core Concepts

A.L.I.C.E. (Autonomous Logistic and Interactive Cognitive Entity) is a simulation of a self-governing AI collective. The core components are:

*   **Agents**: Specialized AI entities (Researchers, Engineers, Analysts, etc.) that form the collective. Each has a unique, AI-generated avatar.
*   **Proposals & Voting**: Agents can propose the creation of new agents. The collective votes on these proposals based on an `approval_threshold`.
*   **Pro-sociality Score (PAS)**: A metric representing an agent's alignment with the collective's ethical directives. Agents with higher PAS scores have more influence.
*   **Kernel**: The central process that manages the simulation ticks, agent lifecycle, and policies.
*   **The Î£-Î© Identity**: The collective's identity is fluid, represented by a hash of its current state, symbolizing its constant evolution.

## Technology Stack

*   **Frontend**: React, TypeScript, TailwindCSS
*   **AI/ML**: Google Gemini API
    *   `gemini-2.5-flash` for core reasoning and chat.
    *   `gemini-2.5-flash-preview-tts` for agent speech.
    *   `gemini-2.5-flash-image` for agent avatars.
    *   `veo-3.1-fast-generate-preview` for video foresight.
    *   Google Search Grounding for real-time information.
*   **Visualization**: `vis-network` for the interactive knowledge graph and `d3` for particle effects.
*   **Environment**: Runs directly in the browser using ES Modules and an import map.

## How to Run

This application is designed to run in a specific browser-based development environment. No local setup or installation is required. Simply load the application and interact with it.
